# Tizi 梯云纵
## 项目简介
项目主要是将购买个人服务器后的一些维护性工作封装成脚本后，从而使得部署环境快速、有效、稳定

> 现在在购买了自己的个人服务器后，很多操作基本都是固化的，下特定软件并安装软件<br>
> 但是存在一些常识性or经验性的东西不太理解，这个带来的就是很多人的工具搭起来了<br>
> 没有维护后，一段时间就被当作肉鸡了，之后就被封了<br>

目前初期弄出来一个检查ssh服务的，实际上很多人的服务器一直是处于被攻击状态的
你不知道而已，有些维护技巧可以大大提升这种安全性，将检查、处理等操作都封装起来

## 功能描述
>[x]服务器暴力破解检查<br>
>[x]服务器ssh服务处理<br>
>[ ]后续再集成一些能主动处理的脚本<br>

## 简要用法：
```
[root@VM_0_6_centos ~]# bash SafeCheck.sh  -c
==================== 开始检查服务 ====================
========》检查项：暴力破解记录
[ info ] 无大量密码爆破尝试记录
检查结果：PASS

========》检查项：SSH端口设置
[ info ] 使用的ssh端口为:11547
检查结果：PASS

========》检查项：SSH密码登录
[ info ] 未启用远程密码登录，无威胁！
检查结果：PASS

检查项完成！检查结果：
检查完毕，所有项检查正常！
```
