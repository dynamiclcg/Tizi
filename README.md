# Tizi 梯云纵
## 项目简介
项目主要是将购买个人服务器后的一些维护性工作封装成脚本后，从而使得部署环境快速、有效、稳定

> 现在在购买了自己的个人服务器后，很多操作基本都是固化的，下特定软件并安装软件
> 但是存在一些常识性or经验性的东西不太理解，这个带来的就是很多人的工具搭起来了
> 没有维护后，一段时间就被当作肉鸡了，之后就被封了

目前初期弄出来一个检查ssh服务的，实际上很多人的服务器一直是处于被攻击状态的
你不知道而已，有些维护技巧可以大大提升这种安全性，将检查、处理等操作都封装起来

## 功能描述
>[x]服务器暴力破解检查
>[ ]服务器ssh服务处理
>[ ]后续再集成一些要
